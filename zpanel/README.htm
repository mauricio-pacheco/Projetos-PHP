[Before Install]

In your PHP.ini file, set the following variables and restart
your web server software.

- max_execution_time = 300
- max_input_time = 300
- Make sure GD2 libraries are runing. (extension=php_gd2.dll)


[To Install]

1. Unzip or Unrar ZPanel and upload it to your server.
2 . CHMOD the zpanel directory to 777 (on *nix) ... Give the directory everything except for "Full Control" (Windows-IIS) ... Skip this step if you're on Windows with Apache.
3 . Open and edit zpanel/database/db.php
4 . Create a new database on your MySQL server where you'd like ZPanel to install to
5 . Go to http://yourdomain/zpanel-directory/admin/install.php and it will guide
you through the rest of the installation.

[To Upgrade]

To upgrade from v1.5 to v2.0

1. Go to http://yourdomain/zpanel-directory/admin/install.php?update=yes&adminpass=yourpass and it will guide
you through the rest of the update.

Note: yourpass is the original password that you entered with you first installed your zpanel.
