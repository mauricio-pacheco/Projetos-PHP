ZPanel Ver. 2.0
Release date: 4/15/2004

For support visit our site at: http://www.thezpanel.com

[v 2.0]

This list is partial because we didn't log every single fix or enhancement.

*Fixed a lot of the minor bugs

*Installed private messaging

*Updating the billing page to a better design

*Installed the ability to use templates (an example/tutorial is in templates/ZPanelV2)

*Added a Mass Mailing feature for admins

*Added an ability for users to change certain aspects of their profile

*Moved the installers to the installers/ folder like they were supposed to be

*Updated all of the installers to their newest versions. (ie. phpNuke 7.2)

*Ported all of the text into a language file. The english language file can be found at zpanel/languages/english.php

[v 1.5.1]

*Fixed "Local path to ZPanel" not showing up in the Administration page.

*Fixed a couple of bugs in the Administration page

*Added an update feature so hosts don't have to re-enter their users. (example in the readme)

[v 1.5]

*Fixed many errors that didn't effect function. Just wanted to make it look cleaner if you have display_errors set to 'on' in your php.ini

*Fixed the "Web Service" drop down when adding a new user. Now it correctly displays all of the services.

*Added an easy mysql injection page into the installer so you don't have to load the database tables using the .sql file. Just trying to make it easier for you!

*By request, you can specify the URL for each user. In case you don't provide them with subdomains.

*Upgraded the filemanager. Fixed filemanager "not allowed ftp server" issue.

*Hid the filemanager icon from IIS hosts. (IIS for some reason won't understand my code and keeps displaying no directories/files.)
