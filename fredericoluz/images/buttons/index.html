O que é PHP-Nuke


PHP-Nuke é um SGC (Sistema de Gerenciamento de Conteúdo), termo advindo do Inglês Content Managment System reconhecido facilmente pela popular sigla, o CMS. O sistema recebe este nome porque integra, todas as ferramentas necessárias para criar e gerenciar um portal, seja ele comercial ou institucional.
